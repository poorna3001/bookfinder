# 📚 Book Finder App

A simple React + Tailwind app to search for books using the Open Library API.

## 🚀 Features
- Search by title
- Display book cover, author, and year
- Responsive UI

## 🧰 Tech Stack
- React
- Tailwind CSS
- Open Library API

## 🧠 How to Run Locally
```bash
npm install
npm run dev
```
Then open the local URL shown in your terminal (usually `http://localhost:5173`).

## 🌐 API Reference
[Open Library Search API](https://openlibrary.org/search.json?title={bookTitle})
